JSON : Javascript Object Notation : 자바 스크립트 객체 표기법

  • 데이터의 요청 및 응답ㅊ퍼리에 필요한 데이터의 표준 형식을 제공하는 타입
  • 백엔드(서버)언어의 종류에 상관없이 데이터 송수신 단일 형식으로 통일되어 사용
  • 데이터 전송은 전부 String 타입으로 이루어짐
  • 서버로 데이터를 보낼 때 > JSON to String > JSON.Stringify (js object)
  • 서버로 부터 데이터를 받을 때 > string to json > JSON.parse (string data)
